Square1 jQuery Slider

Download and check out the documentation on GitHub.

Examples

One thing that sets Square1 apart from other sliders is the ability to either have the images fill the space or be resized so that the entirety of the image is shown without cropping. The fill_mode property will let you control how and how much of your images people see.

The default value is "cover", which resizes and crops images to fill the slideshow fully.

To create a slideshow, simply create a container element filled with images:

<div class="slideshow"> <img src="image1.png" alt="Caption 1"> <img src="image2.png" alt="Caption 2"> <img src="image3.png" alt="Caption 3"> </div>

OR, you can include slides as HTML content elements. The `img` tag inside the slide will be used as the slide's background:

And convert it to a sqaure1 slider by calling the square1() function in your javascript:

$('.slideshow').square1();

You can also make slides out of more complex HTML structures. Square1 will use the first `img` tag in each child element as the background of that slide.

<div class="slideshow"> <div> <img src="image1.png" alt="Caption 1"> <h3>Slide Title</h3> </div> <div> <img src="image2.png" alt="Caption 2"> <h3>Slide Title</h3> </div> <div> <img src="image3.png" alt="Caption 3"> <h3>Slide Title</h3> </div> </div>

The "contain" fill mode will resize each image so that it fits completely inside the slideshow.

$('.slideshow1').square1({ fill_mode: 'contain' });

You can easily control the size of the slideshow and the position/visibility of the UI.

$('.slideshow2').square1({ width: '200px', height: '200px', prev_next_nav: 'outside', dots_nav: 'outside', caption: 'none' });

Transitions can either be set to a cross-fade (default) or a 'slide' animation like this one here:

$('.slideshow_slide').square1({ animation: 'slide' });

Additionally, if you are using the default "cover" fill mode to scale your images, you can set the point from which the image should scale from. You can use any values that work with the CSS [background-position](https://www.w3schools.com/cssref/pr_background-position.asp) property.

$('.slideshow_scalefrom').square1({ scale_from: 'bottom right' });

And you can set this on an image-by-image basis by adding the 'scale-from' attribute to your images (resize your browser to see the result):

<div class="slideshow_scalefrom"> <img src="image1.png" alt="Caption 1" scale-from="center center"> <img src="image2.png" alt="Caption 2" scale-from="top right"> <img src="image3.png" alt="Caption 3"> </div>

Square1 also comes with a 'light' theme for use on darker images

$('.slideshow3').square1({ theme: 'light' });

With built in events and callbacks, you can also control and respond to events in the slideshow however you want.

$('.slideshow4').square1({ auto_start: false, onPlay: function() { $('.status').html('Playing');}, onStop: function() { $('.status').html('Stopped');}, onChange: function() { $('.status').html('Changing slides');}, }); // BUTTONS $('.play').click(function() { $('.slideshow4').square1('play'); }); $('.stop').click(function() { $('.slideshow4').square1('stop'); }); $('.next_slide').click(function() { $('.slideshow4').square1('next'); }); // Jump to Slide 3 $('.slide3').click(function() { $('.slideshow4').square1(3); });
